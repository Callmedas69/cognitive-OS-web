# Hooks

Hooks are what make cognitiveOS zero-maintenance. They run on session start and session end to read and update [`memory.md`](/docs/memory) automatically — so you never run a "save" command.

For Claude Code, `init` generates slash-command hooks: `/project:start`, `/project:end`, and `/project:dump`.

> Full reference — per-agent hook wiring and customization — is coming soon. Codex and Antigravity hook implementations are open [community targets](/docs/contributing).
