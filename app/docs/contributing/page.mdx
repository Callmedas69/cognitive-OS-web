# Contributing

cognitiveOS is MIT and open source. Issues and PRs are welcome.

The highest-value open targets are **hook implementations for Codex and Antigravity** — bringing the same automatic session start/end state updates that Claude Code already has to every agent.

> A full contributing guide is coming soon. In the meantime, open an issue or PR on [GitHub](https://github.com/Callmedas69/cognitive-OS).

See also: [Hooks](/docs/hooks) · [Agent Setup](/docs/agent-setup).
