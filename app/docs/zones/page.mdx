# The Zones

Each folder cognitiveOS generates is one **cognitive mode**. You never decide where to look — your agent routes to the right zone automatically.

| Zone | What it holds |
| --- | --- |
| `brain-dump/` | Capture everything, no filter. |
| `queue/` | What's next, already sorted. |
| `focus/` | The ONE thing right now — one task, enforced. |
| `projects/` | Active work, max 3. |
| `ideas/` | Captured, not committed. |
| `someday/` | Not now, not never. |
| `sessions/` | Auto-logged history. |

> Full per-zone reference — when to use each, how the agent routes between them — is coming soon.
