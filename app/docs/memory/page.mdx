# memory.md

`memory.md` is your persistent brain — the single file your agent reads at the start of every session and updates at the end. It holds your current focus, blockers, and open loops, so context survives between sessions.

You never edit it by hand and you never run a "save" command. [Hooks](/docs/hooks) keep it current automatically.
